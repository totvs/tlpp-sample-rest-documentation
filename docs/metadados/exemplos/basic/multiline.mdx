# Exemplo 05 — `description1` … `descriptionN`

**Arquivo:** `src/rest/03_dedicated_function_doc/basic/multiline.tlpp`  
**Metadados:** `description="[U_rest_sample_05_basic_DOC]"` + chaves numeradas no JSON.

<CodeToOpenAPI
tlpp={`"description1" : "Linha 1",;
"description2" : "",;
"description3" : "Linha 2\\n",;
"description6" : "Linha N";`}
yaml={`description: |-
  Linha 1
  Linha 2
  ...
  Linha N`}
/>

**Conceito:** sequência `descriptionN` — regras em [json-documentacao](/docs/tlpp/rest/metadados/json-documentacao).

:::caution Exemplo didático
O trecho TLPP acima ilustra um **gap** (`description3` → `description6`) para mostrar o YAML resultante. Em produção, **não pule números** — o motor para no primeiro buraco.
:::
