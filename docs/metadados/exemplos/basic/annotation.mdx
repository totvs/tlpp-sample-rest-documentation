# Exemplo 02 — metadados na annotation

**Arquivo:** `src/rest/01_traditional_annotation/extended_annotation.tlpp`  
**Metadados:** `description`, `title`, `params`, `responses` (todos na `@Get`).

<CodeToOpenAPI
tlpp={`@Get(;
  endpoint="/rest/sample/doc/basic/02",;
