# tlpp-sample-rest-documentation

Exemplos TLPP de **Doc Generate** — metadados REST, OpenAPI e `tlpp.doc.generate()`.

## Estrutura

| Pasta | Conteúdo |
|-------|----------|
| `src/rest/` | Código `.tlpp` por abordagem de metadados |
| `docs/metadados/` | Conceito — metadados REST |
| `docs/doc-generate/` | Conceito — geração OpenAPI |
| `docs/exemplos/` | Catálogo GitHub (`npm run docs:exemplos`) |
| `docs/sidebar.json` | Menu deste módulo |
| `output/` | Snapshot `api_doc_8080.yaml` |

## Scripts

```bash
npm run docs:exemplos
```
